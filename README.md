# Лабораторная работа 5. Создание Telegram-бота с поддержкой контекста на основе локальной LLM

В файле bot.py представлен код для взаимодействия с LLM посредством общения через Telegram-бота.

## Инструкция по запуску
- Откройте файл bot.py
- Замените API_TOKEN на ваш токен от BotFather
- Запустите bot.py
- Запустите сервер с моделью в LM Studio

## Команды бота
- **/start** - запускает бота и отправляет приветственное сообщение
- **/model** - получение информации о модели
- **/clear** - очищение истории диалога
- Чтобы отправить запрос модели просто введите сообщение с вашим запросом
